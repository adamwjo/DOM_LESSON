# DOM_LESSON

<h3>How to start:</h3>

<h5>1.) Clone and navigate into repo</h5>

<h5>2.) In your terminal, enter these commands:</h5>

``` $ npm install ``` </br>
``` $ npm run server```

<h5>3.) Using your browser navigate to http://localhost:3000</h5>
